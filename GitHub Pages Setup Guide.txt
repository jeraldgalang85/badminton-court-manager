STEP 1: PREPARE YOUR FILE
═══════════════════════════════════════════════════════════════════

1. Find your file: "Badminton Shuffler Mobile.html"
   Location: C:\Users\jjgalang\Documents\Personal\Badminton shuffler\

═══════════════════════════════════════════════════════════════════
STEP 2: CREATE A GITHUB REPOSITORY
═══════════════════════════════════════════════════════════════════

1. Go to: https://github.com

2. Log in with your GitHub account
   Username: jeraldgalang85

3. Click the "+" button in the top-right corner

4. Select "New repository"

5. Fill in the repository details:
   - Repository name: badminton-shuffler
   - Description: Mobile badminton team shuffler app
   - Make it PUBLIC (important for GitHub Pages)
   - DO NOT add README, .gitignore, or license
   - Click "Create repository"

═══════════════════════════════════════════════════════════════════
STEP 3: UPLOAD YOUR FILE
═══════════════════════════════════════════════════════════════════

You'll see a page with several options. Follow these steps:

1. Look for the section that says "Quick setup"

2. Scroll down and find the link "uploading an existing file"

3. Click that link

4. You'll see a page with a file upload area

5. Drag and drop your "index.html" file into the upload area
   OR click "choose your files" and select it

6. Scroll down to the "Commit changes" section

7. In the text box, type: "Add Badminton Shuffler Mobile app"

8. Click the green "Commit changes" button

═══════════════════════════════════════════════════════════════════
STEP 4: ENABLE GITHUB PAGES
═══════════════════════════════════════════════════════════════════

1. Click on "Settings" tab (at the top of your repository)

2. In the left sidebar, scroll down and click on "Pages"

3. Under "Source", you'll see a dropdown that says "None"

4. Click on it and select "main" (or "master")

5. Keep the folder as "/ (root)"

6. Click "Save"

7. Wait a moment... GitHub is setting up your site!

8. Refresh the page after 30 seconds

9. You'll see a message at the top:
   "Your site is published at: https://jeraldgalang85.github.io/badminton-shuffler/"

10. Click on that URL to test your app!

═══════════════════════════════════════════════════════════════════

═══════════════════════════════════════════════════════════════════

If you want to update the app in the future:

3. Click the pencil icon (Edit this file)
4. Make your changes
5. Scroll down and click "Commit changes"
6. Wait 1-2 minutes for changes to go live

═══════════════════════════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

If the site doesn't work:
- Make sure the file is named exactly "index.html" (lowercase)
- Make sure the repository is PUBLIC
- Wait a few minutes after enabling Pages
- Clear your browser cache and try again

If you see a 404 error:
- Double-check the URL spelling
- Make sure GitHub Pages is enabled in Settings > Pages
